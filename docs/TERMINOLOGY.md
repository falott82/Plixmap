# Glossario IT/EN

Questo glossario serve a mantenere coerente la terminologia tra italiano e inglese.
Usa sempre queste traduzioni quando scrivi nuove stringhe UI o documentazione.

## Linee guida
- Mantieni la stessa capitalizzazione e gli stessi plurali.
- Se un termine UI e' gia' tradotto qui, non inventare varianti.
- Per stringhe nuove, aggiungi la voce qui prima del merge.
